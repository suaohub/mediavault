# MediaVault

一款基于浏览器的本地媒体库管理工具，支持视频和音频文件的管理、预览和播放。

## 功能特性

- **文件导入** - 支持拖拽导入、文件选择、文件夹导入
- **目录连接** - 连接本地目录，自动恢复上次导入
- **媒体预览** - 鼠标悬停自动播放视频预览
- **播放控制** - 支持倍速播放 (1x/1.5x/3x/4x)
- **进度记忆** - 自动保存播放位置，下次恢复
- **智能排序** - 支持按导入时间、名称、大小、时长排序，随机打乱
- **网格调整** - 可调节卡片大小，改变视口显示数量
- **主题切换** - 6 种主题配色可选
- **键盘导航** - 方向键快速浏览，快捷键控制播放
- **收藏功能** - 收藏喜欢的视频/音频

## 支持的格式

- **视频**: MP4, MOV, MKV, AVI, WEBM, M4V, WMV, FLV, TS, M2TS
- **音频**: MP3, FLAC, WAV, AAC, OGG, OPUS, M4A, WMA, AIFF

## 快速开始

### 方式一：直接打开
1. 下载项目文件
2. 用浏览器直接打开 `index.html`
3. 点击"导入"按钮选择媒体文件

### 方式二：使用本地服务器
```bash
# 使用 Python
python -m http.server 3000

# 或使用 Node.js
npx serve .
```

然后访问 `http://localhost:3000`

## 快捷键说明

### 列表浏览
- `↑↓←→` - 方向键移动选中（支持对角线）
- `Enter` - 打开选中的视频/音频
- `Ctrl + Enter` - 换一批视频，优先显示上一批没有的视频
- `Esc` - 关闭播放器

### 播放器控制
- `Space` - 播放/暂停
- `← →` - 快退/快进 5 秒
- `↑ ↓` - 调节音量
- `N / P` - 下一个/上一个
- `F` - 收藏/取消收藏
- `M` - 静音切换
- `Esc` - 关闭播放器

## 技术栈

- 纯原生 JavaScript (ES6+)
- 零依赖，无需构建
- 使用 IndexedDB 缓存媒体元数据
- CSS 变量实现主题切换
- File System Access API (可选)

## 浏览器兼容性

- Chrome / Edge (推荐)
- Firefox
- Safari (部分功能受限)
